# 개발프로젝트

[사용자 보행속도 기반 길 찾기 앱 개발]

  1. 진행 기간 및 규모 : 2020.04 ~ 2020.05 (약 2개월), 개발 2명(본인 포함)
  2. 주요 내용 : GoogleMap, MapMyWalk, 'Odsay' 등 8개가 넘는 open API를 활용해 사용자 보행속도에 따른 길 찾기 및 출발시간 (여유, 촉박) 예측 서비스 개발
  3. 본인이 공헌한 점 : 프로젝트 기획, 설계, 디자인, 풀스택 개발
  4. 사용 기술 : React Native/CSS/JS 기반 프론트, Node.js 기반 서버, Mysql
  5. 결과/성과 : 서버 클라이언트간의 상호작용, Rest에 대한 이해, Ajax 비동기 통신에 대한 이해, react-native 습득

[AutoEncoder, CNN 기반의 화가에 따른 Style(명화) 분류 가능한 딥러닝 모델 제안]

  1. 진행 기간 및 규모 : 2019.10 ~ 2019.12 (약 2개월), 개발 1명(본인)
  2. 주요 내용 : 기존에는 Resnet, Vggnet 등의 여러 학습 모델이 개미, 벌과 같이 무언가 형상을 띠는 image를 분류하는 연구가 많았습니다. 하지만 style이나 texture를 분류에 해당 모델을 적용할 경우 정확도가 확연히 낮았습니다. style, texture를 분류한다는 말이 이상할 수 있지만 우리는 앤디워홀의 작품과 반 고흐의 작품이 서로 다르다고 분류할 수 있습니다. 그래서 명화의 색감이나 스타일의 특징을 잘 추출해 서로 다른 화가가 그린 그림이라 분류할 수 있는 새로운 딥러닝 모델을 구현했습니다.
  3. 본인이 공헌한 점 : 딥러닝 모델 구현, 모델 정확도 향상을 위한 하이퍼 매개변수 조정
  4. 사용 기술 : Python, Pytorch
  5. 결과/성과 : AutoEncoder, classifier mixture에 대한 이해 및 습득, batch size, learning rate 에 따른 모델 성능 좌우에 대한 이해

[KIV 월드프렌즈 ICT 봉사단 우샤히디 오픈소스를 활용한 크라우드 소싱으로 콜레랑 예방 웹 개발]

  1. 진행 기간 및 규모 : 2018.08 ~ 2018.08 (약 1개월), 개발 4명(본인 포함)
  2. 주요 내용 : 문자, 트위터 등으로 취합된 쓰레기 매립 지역 정보를 실시간으로 받아 시각화해 콜레라 예방을 위한 웹 개발
  3. 본인이 공헌한 점 : 개발일지 작성, 로컬환경에서 우사히디 오픈소스 작동, AWS EC2 작동 시도, 테스트 작업
  4. 사용 기술 : APM(Apache + PHP + Mysql)
  5. 결과/성과 : 구체적인 개발일지 작성의 중요성, 오픈소스 사용을 위한 시스템 환경구성(버전)의 중요성 경험, Linux 개발 환경 경험

[펀딩상품 정보 및 소비자 구매 내역 분석을 통한 크라우드 펀딩 상품 추천]

  1. 진행 기간 및 규모 : 2020.04 ~ 2020.06 (약 3개월), 개발 4명(본인 포함)
  2. 주요 내용 : 1) 와디즈 펀딩 상품 및 소비자 구매 내역 크롤링해서 데이터 추출 2) 해당 데이터로 Content Based Filtering, Collaborative Filtering으로 분석하여 펀딩 상품 추천 3) 추천 결과를 서로 다른 알고리즘을 기준으로 유사도 비교
  3. 본인이 공헌한 점 : 웹 크롤링을 통한 데이터 추출, 크롤링 코드 최적화, 데이터 전처리, Cotent Based Filtering 추천 알고리즘 구현(CosineSimilarity)
  4. 사용 기술 : Jupyter Notebook, Python, hadoop, spark, Node.js(express), ejs
  5. 결과/성과 : 대용량 데이터 분산 처리 경험, 크롤링 시간 단축을 위한 코드 최적화의 중요성 경험, Spark RDD / DataFrame 문법 습득
